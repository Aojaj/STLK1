# STLK - TikTok Profile Website

Website modern untuk menampilkan profil TikTok Anda dengan desain yang elegan dan responsif.

## 🌟 Fitur

- ✨ **Desain Modern**: Interface yang sleek dengan gradient dan efek blur
- 📱 **Responsive**: Sempurna untuk desktop, tablet, dan mobile
- 🎬 **Showcase Videos**: Tampilkan video TikTok Anda dengan statistik
- 🔗 **Social Links**: Link ke semua platform media sosial
- 🎨 **Animasi Smooth**: Loading animations dan hover effects
- ⚡ **Lightning Fast**: Website yang ringan dan cepat

## 🚀 Cara Menggunakan

1. Clone repository ini atau download files
2. Buka `index.html` di browser Anda
3. Sesuaikan data profil di `index.html`:
   - Username TikTok
   - Bio/Deskripsi
   - Social media links
   - Foto profil dan video thumbnails
4. Deploy ke GitHub Pages atau hosting lainnya

## 📝 Cara Customize

### Mengubah Informasi Profil
Edit file `index.html`:
```html
<h1 class="profile-name">STLK</h1>
<p class="profile-username">@stlk_official</p>
<p class="profile-bio">🎬 Content Creator | 🎵 Music Lover | ✨ Follow for more</p>
```

### Mengubah Statistik
```html
<span class="stat-number" id="followers">1.2M</span>
<span class="stat-number" id="likes">45M</span>
<span class="stat-number" id="videos">256</span>
```

### Mengubah Social Links
Ganti URL di bagian social links:
```html
<a href="https://www.tiktok.com/@your_username" target="_blank">
```

### Mengganti Foto dan Gambar
Ganti URL placeholder dengan foto Anda:
```html
<img src="https://via.placeholder.com/150" alt="Profile Picture">
```

## 🎨 Warna dan Styling

Edit `style.css` untuk mengubah warna:
```css
:root {
    --primary: #25f4ee;     /* Cyan (TikTok)*/
    --secondary: #fe2c55;   /* Pink (TikTok) */
    --dark: #0a0e27;        /* Background gelap */
    --light: #ffffff;       /* Teks putih */
    --gray: #666666;        /* Gray */
}
```

## 📦 Teknologi

- HTML5
- CSS3 (dengan Flexbox & Grid)
- Vanilla JavaScript
- Font Awesome Icons

## 🌐 Deploy ke GitHub Pages

1. Push repository ke GitHub
2. Buka Settings > Pages
3. Pilih branch `main` atau `master`
4. Tunggu deployment selesai
5. Akses melalui `https://username.github.io/STLK1`
